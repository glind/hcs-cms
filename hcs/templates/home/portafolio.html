Portafolio

Null Records

Null Records ayuda a los artistas independientes a conectarse con fanáticos y patrocinadores como nunca antes. El poder colectivo de un sello organizado como colectivo de artistas visuales y musicales gestionando sus propios canales y su propio destino.


Activity (TolaData Community Edition)

Activity es una bifurcación del repositorio TolaData Community Edition compilada originalmente para Mercy Corps por el equipo de Open Build. Es una solución elegante y fácil de usar para que las organizaciones sin fines de lucro la utilicen para administrar, monitorear y rastrear actividades, tareas y datos de programas y proyectos.


CMS

CMS es un sistema de gestión de contenido simplificado creado por Open Build utilizando bibliotecas de código abierto para ser utilizado como base no solo para un sitio web de folleto o blog, sino como interfaz funcional y conectada a datos para cualquiera o todas sus aplicaciones.
